# Cipher-Chatbot
A chatbot application named Cipher made to provide educational insight to cybersecurity awareness

This was a project made in my second year of my Software Development Diploma. The aim was to create a chatbot application that educated people through an interactive TUI about the dangers of cybersecurity all with the goal to make people aware of the dangers they could face.
The assignment tested our ability to utilize the C# programming language to solve problems. This is my assignment

# How To Run
You need a C# interpreter and run the program.cs file
the program has built in error handling such as blank inputs as well as only being limited to only being able to answer questions related to cybersecurity.
Attatched is also some research conducted on the topic (ChatBot Application Resources.pdf)
